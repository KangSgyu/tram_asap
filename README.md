[물리 기반 가상환경과 심층 강화학습을 이용한 제조 로봇 시뮬레이션 확장안]

1. 연구 목적
 
 본 연구의 확장 시뮬레이션은 기존의 단순 궤적 추종 기반 제조 로봇 검증과 로봇이 물리 법칙이 적용되는 가상 제조 셀 안에서 센서 관측을 해석하고 보상 신호를 통해 행동 정책을 학습하는 구조를 제안한다. 이를 통해 중소 제조 현장에서 실제 로봇을 반복 시험하기 전에 충돌, 물체 접촉, 마찰, 중력, 센서 잡음, 작업 실패 가능성을 안전하게 검증할 수 있는 Physical AI 기반 사전 학습 환경을 구성한다.


2. 가상 물리 환경 구성
 
 가상환경은 테이블, 엔드이펙터, 조립 대상 물체, 목표 위치, 장애물로 구성된 탁상형 제조 셀로 모델링하였다. 물리 엔진은 경량 강체 동역학 모델을 사용하며, 각 시간 단계마다 다음 물리 현상을 계산한다.

- 중력: 물체가 파지되지 않은 상태에서 수직 방향 가속도를 적용한다.
- 마찰: 물체가 테이블과 접촉할 때 수평 속도를 감쇠시켜 재료와 작업대 사이의 저항을 반영한다.
- 충돌: 로봇 엔드이펙터가 물체 또는 장애물과 접촉할 때 반발계수와 접촉 강성을 적용한다.
- 재료 순응성: 강체, 반강체, 유연체 공정에 따라 물체가 로봇 명령에 반응하는 정도를 다르게 설정한다.
- 안전 제한: 로봇 손목 위치는 작업 공간과 최대 도달 범위 안으로 제한하여 비현실적인 동작을 방지한다.

이러한 구성은 고정밀 상용 물리 엔진을 대체하기 위한 최종 모델이라기보다, 논문 실험 단계에서 반복 가능한 정책 학습과 정량 비교를 수행하기 위한 실행 가능한 물리 근사 환경이다.


3. 센서 시뮬레이션
 
 로봇 정책은 정답 상태를 직접 사용하는 것이 아니라, 실제 로봇 제어에 가까운 다중 센서 관측 벡터를 입력으로 받는다. 본 연구에서는 다음 세 종류의 센서를 가상으로 생성하였다.

- 카메라: 상부 시점의 저해상도 점유 이미지를 생성하여 엔드이펙터, 물체, 목표점, 장애물의 상대 위치를 표현한다. 이미지에는 잡음을 추가하여 실제 비전 센서의 불확실성을 반영한다.
- 라이다: 로봇 기준 평면 방향으로 여러 개의 거리 광선을 발사하여 물체와 장애물까지의 정규화된 거리를 산출한다.
- IMU: 엔드이펙터 속도 변화와 행동 벡터의 회전 성분을 이용해 선형 가속도와 각속도에 해당하는 6채널 관측값을 만든다.

최종 관측값은 로봇 고유 상태, 물체 상태, 목표 위치, 카메라 특징, 라이다 거리, IMU 신호를 결합한 벡터로 구성된다. 이는 로봇이 단순 좌표 추종이 아니라 센서 기반 상태 인식을 통해 행동을 선택하도록 하기 위한 설계이다.
